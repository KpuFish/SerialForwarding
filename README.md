# SerialForwarding
> 2개의(Host/Remote) 시리얼통신을 서로 연결해주는 유틸리티 


> Host 와 Remote 의 송수신 메시지를 서로 연결 

## Develop Tools
python 2.7 

pyqt 4.10

## Build Setup
``` bash
# run shell
python SF_main.py

# Windows Exe 
pyinstaller.exe --noconsole --onefile SF_main.py
```
