# SerialForwarding
> Utility to connect two serial communications.

## Develop Tools
python 2.7 
pyqt 4.10

## Build Setup
``` bash
# run
python SF_main.py

# generate exe
pyinstaller.exe --noconsole --onefile SF_main.py
```
